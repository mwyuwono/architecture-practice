# Agent Instructions

This file provides guidance for AI coding assistants working with this repository.

## Primary Instructions

**All AI agents should read and follow the instructions in [`CLAUDE.md`](CLAUDE.md).**

The CLAUDE.md file contains:
- Project overview and purpose
- Design system integration guidelines
- Code style and quality standards

## Quick Reference

- **Design System**: M3 Design System v2 (`m3-design-v2`)
- **Component Prefix**: `wy-*` web components
- **Aesthetic**: Soft Modernism (organic shapes + editorial typography + warm palette)
- **CSS Rule**: Always use design tokens, never hardcode values
